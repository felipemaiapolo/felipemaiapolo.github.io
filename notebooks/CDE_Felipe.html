Estimando Densidade Condicional¶

Felipe Maia Polo (felipemaiapolo@gmail.com)¶

Suponha um conjunto de dados $\mathcal{D}=\{(x_i,y_i)\}_{i=1}^n$ amostrado de forma independente de uma distribuio $P_{X,Y}$ com densidade $p_{X,Y}$. Nosso objetivo estimar a densidade condicional $p_{Y|X}$ a partir do conjunto de dados $\mathcal{D}$. Suponha que $Y$ seja uma varivel aleatria que assume valores em $\mathcal{Y}=[y_\text{min},y_\text{max}]$ com probabilidade 1, sendo que $\Delta y:=y_\text{max}-y_\text{min}$. Afim de continuarmos, considere a seguinte varivel aleatria 'dummy' $\tilde{Y}\sim U[y_\text{min},y_\text{max}]$ independente de $(X,Y)$ e com f.d.p. dada por $p_\tilde{Y}(y)=\frac{1}{\Delta y}\mathbb{I}_\mathcal{Y}(y)$. Agora veja que podemos reescrever $p_{Y|X}$ da seguinte forma:

\begin{align} p_{Y|X}(y|x)&=\frac{p_{X,Y}(x,y)}{p_{X}(x)} \\[.5em] &=\frac{\Delta y}{\Delta y}\frac{p_{X,Y}(x,y)}{p_{X}(x)} \\[.5em] &=\frac{1}{\Delta y}\frac{1}{p_\tilde{Y}(y)}\frac{p_{X,Y}(x,y)}{p_{X}(x)} \\[.5em] &=\frac{1}{\Delta y}\frac{p_{X,Y}(x,y)}{p_{X,\tilde{Y}}(x,y)} \end{align}

Agora considere o vetor aleatrio $(X',Y')\sim P_{X',Y'}$ com f.d.p. $p_{X',Y'}$ e uma varivel aleatria $S \sim \text{Bernoulli}(1/2)$, sendo que $p_{X,Y}(x,y)=p_{X',Y'|S}(x,y|S=1)$ e $p_{X,\tilde{Y}}(x,y)=p_{X',Y'|S}(x,y|S=0)$. Ou seja, $S$ uma varivel que indica se uma amostra vem de $P_{X,Y}$ ou de $P_{X,\tilde{Y}}$. Logo:

\begin{align} p_{Y|X}(y|x)&=\frac{1}{\Delta y}\frac{p_{X',Y'|S}(x,y|S=1)}{p_{X',Y'|S}(x,y|S=0)} \\[.5em] &= \frac{1}{\Delta y}\frac{\mathbb{P}_{S|X',Y'}(S=1|x,y)}{\mathbb{P}_{S|X',Y'}(S=0|x,y)} \end{align}

A partir dos dados $\mathcal{D}$ criamos uma base de dados 'dummy' $\tilde{\mathcal{D}}=\{(x_i,\tilde{y}_i)\}_{i=1}^n$, com os pontos $(\tilde{y}_i)_i$ sendo realizaes de $\tilde{Y}$. Criamos rtulos artificiais para as amostras de $\mathcal{D}$ e de $\mathcal{D}'$, sendo que o primeiro recebe rtulos 1 e o segundo recebe rtulos 0. O nosso estimador para a densidade condicional dada da seguinte maneira

\begin{align} \widehat{p}_{Y|X}(y|x)&=\frac{1}{\Delta y}\frac{\widehat{\mathbb{P}}_{S|X',Y'}(S=1|x,y)}{\widehat{\mathbb{P}}_{S|X',Y'}(S=0|x,y)} \end{align}

Sendo que $\widehat{\mathbb{P}}$ representa um classificador probabilstico no-linear (regresso logstica combinada com uma expanso das variveis originais, XGBoost, Rede Neural MLP, etc). Em situaes prticas podemos no conhececer os valores $y_\text{min},y_\text{max}$, mas podemos adotar $y_\text{min}=\text{min}\{y_i\}_{i=1}^n$ e $y_\text{max}=\text{max}\{y_i\}_{i=1}^n$ se no temos informaes o suficiente a priori.

def cde(x):
  delta_y=y_max-y_min
  z=np.vstack((np.array(100*[x]),np.linspace(y_min,y_max,100))).T
  z=poly.fit_transform(z)
  return model.predict_proba(z)[:,1]/(delta_y*model.predict_proba(z)[:,0])
